## Overview
This assignment covers both theoretical and practical aspects of machine learning and neural networks.

## Main Topics
- Neural network representations and ReLU activation
- Softmax function and cross-entropy derivatives
- Implementation of activation functions and their gradients

- CNN implementation for image classification (Big Cats dataset)

- Saved trained weights for reuse
