##  Overview

This repository contains a collection of projects developed as part of advanced machine learning courses.

The projects cover a wide range of topics, from classical neural networks to modern deep learning techniques including sequence modeling, adversarial attacks, and generative AI.

Each project demonstrates both **theoretical understanding** and **practical implementation from scratch**, with a strong emphasis on model design, training, and evaluation.
